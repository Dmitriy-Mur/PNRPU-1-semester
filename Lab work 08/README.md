# Задание
Реализовать структурау "Студент" с полями:
- фамилия, имя, отчество
- дата рождения
- домашний адрес
- рейтинг

Выполнить задание над структурой:
- создать массив структур
- удалить элементы, у которых даты рождения совпадают
- добавить элемент перед элементом с заданной фамилией
